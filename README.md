## 🚀 Featured Project

### 🌊 [HealthLakes](https://healthlakes.com)
> Geospatial web platform for environmental simulation, commissioned by the **Provincial Watershed Authority of Córdoba, Argentina**.

Models the impact of residential developments on lake water quality (Phosphorus, Nitrogen, BOD5) using interactive maps, raster processing, and hydrological models developed by PhD researchers.

**What I built end-to-end:**
- 🔧 **Backend:** FastAPI · Python · SQLAlchemy + Alembic · PostgreSQL + PostGIS · Celery · Redis
- 🎨 **Frontend:** React · TypeScript · Leaflet · TailwindCSS · Vite
- ☁️ **Infrastructure (AWS):** CloudFront · EC2 · RDS · S3 · ECR · SSM · CloudWatch · VPC/IAM
- 🔄 **CI/CD:** GitHub Actions · multi-repo pipeline · automatic rollback · staging + prod environments
- 👥 **Team lead:** coordinated software engineers, administrative staff, and PhD scientists
